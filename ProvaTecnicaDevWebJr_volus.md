## Critérios de Avaliação

### 1. Frontend Avançado (50 pontos)

#### Desafio 1: Dashboard Interativo e Responsivo (25 pontos)

##### Objetivo
Desenvolver um dashboard administrativo com componentes interativos e design responsivo.

##### Requisitos
- Layout responsivo
- Design mobile-first
- Breakpoints personalizados

##### Especificações Técnicas
- Utilize Chart.js ou Highcharts para visualização de dados
- Implemente transições e animações sutis
- Adicione tooltips personalizados

##### Desafios Especiais
1. Gráfico Dinâmico
   - Dados que mudam sutilmente ao passar o mouse
   - Total aparente constante
   - Requer análise cuidadosa

2. Menu Lateral Interativo
   - Submenu com comportamento não intuitivo
   - Animações de hover complexas
   - Ícones com comportamento inesperado

3. Validações de Formulário
   - Regras de validação não óbvias
   - Máscara de input personalizada
   - Feedback visual único

##### Critérios de Avaliação
- Criatividade na implementação
- Responsividade
- Interatividade
- Tratamento de casos edge
- Performance frontend

#### Desafio 2: Componente de Filtro Dinâmico (25 pontos)

##### Requisitos
- Filtro multiníveis customizado
- Uso intensivo de jQuery para manipulação do DOM
- Carregamento dinâmico sem refresh

##### Especificações
1. Filtro Cascata
   - Mínimo 3 níveis de seleção
   - Atualização em tempo real
   - Animações de transição

2. Funcionalidades Avançadas
   - Seleção múltipla com chips
   - Pesquisa em tempo real
   - Resetar filtros com animação

3. Desafios de Performance
   - Otimização de filtragem
   - Minimizar reflows e repaints
   - Implementar lazy loading

##### Desafios Extras
- Filtro "coringa" com comportamento inesperado
- Lógica de filtro com regra não documentada
- Easter egg visual ao combinar filtros

### 2. Backend e Integração (30 pontos)

#### Desafio 3: Integração Django (15 pontos)

##### Requisitos

##### Desafios Backend

- Tratamento de exceções
- Implementação de filtros básicos
